Nike Sneaker Sales Analytics Dashboard
Overview:
Created an interactive sales dashboard analyzing Nike sneaker sales data to visualize key business metrics and performance trends.
Key Metrics Tracked:

Total Sales Amount: ₹113.33M
Total Cost: ₹71.23M
Quantity Sold: 24.05K units
Total Transactions: 7.83K
Profit Margin: 37.2%

Insights Delivered:

Category Performance: Single Monk shoes lead in quantity (35.3%), followed by Derby Shoes (13.6%) and Sneakers (17.2%)
Order Type Analysis: Breakdown of Casual, Formal, and Semi-Casual shoe sales
Temporal Trends: Monthly sales patterns showing peak performance in March (₹20.9M) and lowest in July (₹1.8M)
Geographic Distribution: State-wise profit analysis with Uttar Pradesh (₹5.2M) and West Bengal (₹5M) as top performers
Visual Heat Map: Geographic visualization of sales distribution across India

Tools Used:

Google Sheets for data processing and dashboard creation
Interactive filters for Category, State, Order Type, City, and Customer Name

Features:

Dynamic filtering capabilities
Geographic heat map visualization
Multi-dimensional sales analysis
Real-time metric calculations
